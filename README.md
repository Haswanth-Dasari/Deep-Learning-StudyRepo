# Deep-Learning-StudyRepo
Studying deep learning and implementing neural network architectures while learning from the MIT Introduction to Deep Learning course.

# Deep Learning Projects

This repository documents my study and experimentation while learning deep learning.
The goal is to build a strong conceptual and practical understanding of neural networks by combining theoretical notes with hands-on implementations.

The implementations are written primarily using **TensorFlow** and **NumPy**, and focus on understanding how neural network components work internally rather than only using high-level APIs.
