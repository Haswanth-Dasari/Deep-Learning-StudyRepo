# Neural Network Implementation using NumPy

## Objective

The objective of this project is to understand how a neural network works internally by implementing it from scratch using NumPy.

Instead of relying on deep learning frameworks, this implementation focuses on the mathematical operations that form the foundation of neural networks.

## Concepts Covered

* Forward propagation
* Weight matrices and bias vectors
* Activation functions
* Basic training mechanics

## Architecture

Input Layer → Hidden Layer → Output Layer

Activation Function: Sigmoid

This project demonstrates how neural networks perform computations using matrix operations.
