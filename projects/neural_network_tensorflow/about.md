# Neural Network Implementation using TensorFlow

## Objective

This project explores the implementation of neural networks using TensorFlow.

The goal is to understand how deep learning frameworks manage layers, parameters, and forward propagation.

## Concepts Covered

* TensorFlow tensors
* Custom neural network layers
* Sequential model construction
* Activation functions

## Approach

Two approaches are explored:

1. Custom layer implementation using TensorFlow's Layer subclass.
2. Building neural networks using the Sequential API.
